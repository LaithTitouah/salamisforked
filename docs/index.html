Data. Simulation. Visualization

A small programming language for statistical thinking.

Salamis is a statically typed language designed for working with numbers, vectors, distributions, simulations, csv data, and simple visualization.

View GitHub Repo

The Story

Salamis was created to make statistical and data-centered programming feel lightweight, readable, and beginner friendly. Instead of writing long Javascript programs just to sample from distributions, calculate averages, or graph data, Salamis gives users a focused syntax for exploring numerical ideas quickly.

The language comines familiar programming tools,such as variables, functions,loops, conditionals, and static types with built-in support for vectors,matrices, probability distributions, simulation, plotting, and histograms.

Language Features

  • Static types: Int, Float, Bool, Str, Void, Vec, Matrix, and distributions
  • Variables, assignment, functions, parameters, and return values
  • If / else-if / else conditionals
  • Range loops, collection loops, and while-style loops
  • Vector literals and element-wise vector arithmetic
  • Matrix multiplication
  • Built-in statistical functions like mean, sum, min, and max
  • Probability distributions: Normal, Bernoulli, Poisson, and Uniform
  • Simulation expressions using repeated samplings
  • Plot and histogram generation through html output
  • CSV reading and column extraction

Static, Safety, and Security

  • Variables must be declared before use
  • Variables cannot be redeclared in the same scope
  • Assignments must match the declared variable type
  • Function calls must use the correct number of arguments
  • Return statements must appear inside functions
  • Return values must match the declared function return type
  • If and while conditions must be Bool
  • Arithmetic expressionss must use valid numeric or vector operands
  • Matrix multiplication requires matrix operands
  • Plot and histogram statements require vector output
  • sample(...) only accepts distribution values
  • simulate(...) requires an Int count and a numeric body expression

3. Monte Carlo Simulation


                    let results = simulate(2000) {
                        sample(Normal(0.0,1.0))
                    } 
                print("=== Normal(0,1) - 2000 samples ===")
                print("Mean: "+ format(mean(results), 4))
                histogram(results)

4. Descriptive Statistics and Normalization


                let data: Vec<Float> = [2.0, 4.0, 4.0, 4.0, 5.0, 7.0, 9.0]
                let avg = mean(data)
                let lo = min(data)
                let hi = max(data)
                let normalized = (data - lo) / (hi - lo)
                plot(data)
                plot(normalized)

5. CSV Data Analysis 


                    let m = readCsv("prices.csv")

                    let avg_price = mean(prices)
                    let lo_price = min(prices)
                    let hi_price = max(prices)

                    let norm_prices = (prices - lo_price) /(hi_price - lo_price)
                    plot(prices, norm_prices)
